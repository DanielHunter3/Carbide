### **Циклы

Циклы: for, while, loop, while let


```rust
// `while`
let mut i = 0;
while i < 10 {
    println!("{}", i);
    i += 1;
}

// `for` (итерирование)
for item in [1, 2, 3] {
    println!("{}", item);
}

// Бесконечный цикл
loop {
    println!("Hello");
    break; // Выход
}
```
