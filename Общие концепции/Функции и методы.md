### **Функции**

```rust
fn add(a: i32, b: i32) -> i32 {
    return a + b;
}

// Аналог `Option` (вместо NULL)
fn safe_div(a: f64, b: f64) -> Option<f64> {
    return match b {
	    0.0 => None,
	    _ => a / b,_
    };
}
```
